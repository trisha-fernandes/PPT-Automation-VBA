# Automated PPT via Excel using VBA

With the help of Claude, I created this project to help automate recurring PPTs. 

By just 
- updating the data in the excel file and
- using the template PPT

The PPT can be created and the charts in it would be updated in the click of a button

## How to use
**1.** Download both files and save in the same folder

**2.** Click on the 'Create PPT' button in the first tab on the excel SampleWorkbook file

**3.** You can then update data in the excel file, go through the VBA code and take help of AI to add/update features

## Best use case
Ideal when creating multiple PPTs for different regions or periods. Thus, using slicers, the data can be quickly filtered and the PPT can be created in no time.
